HOW TO COMPILE THE APPLICATION

Add the folder actionbar\library as a project in eclipse (Excisting android application project). Select properties for the project, and make sure "is library" is checked under the "Android" tab.
Also right click on the project, select Android tools -> Add support library.

Add the complete git folder "Gym-app" as an excisting android application project. Select properties again and under the Android tab add the library you just imported.
