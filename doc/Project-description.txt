Beskrivning:
	Appen skall fungera som en träningsdagbok där man lägger till träningspass,
	när man gjort ett moment tar man bort den ur listan. Detta registreras sen
	så man kan se hur mycket man tränat

Licens:
	Vi har bestämt att vi skall använda GPL v3 för att:
	Möjlighet att programmet lever vidare

Features:
	1. möjlighet att skapa träningspass(set, rep, tid, stor knapp att ta tid med)
	2. lägga till moment i träningspass under träning
	3. se alla utförda träningspass
	4. se hur mycket man körde förra gången på specifik övning
	5. olika statistiker för träning
	6. se maxvikt(CONGRATULATIONS)
	7. färdiga träningspass
	8. dela pass
	9. lägga till pass i kalender
	10. Statistik över kroppsvikt. 
	11. Se muskelstatus
